# The Amelia Chat Workspace Walkthrough

## Introduction

The default Amelia chat workspace lets you interact with Amelia by typing or talking. You also can view live data about a conversation as it unfolds.

## Accessing the Amelia Chat Workspace

To access Amelia's chat workspace, click the **Amelia link**. It's located in the top left main navigation menu.

## Use the Amelia Chat Workspace {% #ChatWorkspace %}

The default Amelia chat interface has two panels:

* The left-side panel displays conversation with tools to manage the conversation.

* The right-side panel displays data about a conversation as it happens.


## Use the Chat Input Box {% #ChatInputBox %}

A conversation panel displays on the left side of the workspace. It displays a conversation as it happens. And includes an input field at the bottom to start, interact, and manage a conversation.

> The chat input box also displays a thumbs up and thumbs down button during and after a conversation. This customer satisfaction data helps assess and optimize Amelia's effectiveness.

## Use the Right-Side Panels {% #RightSidePanels %}

During a conversation, a vertical bar displays on the right edge of the chat workspace. The bar includes buttons to display data about the conversation. For example, the Info panel can display user information. And the Journey Analytics button displays process flow details.

You can expand or contract the size of panels that display, as well as close a panel.

* Click the Expand (![Btn-RightPanel-Expand.png](Btn-RightPanel-Expand.png){width="25" style="inline"} ) button to expand the width of a panel.

* Click the Restore (![Btn-RightPanel-Restore.png](Btn-RightPanel-Restore.png){width="25" style="inline"}) button to restore the panel to its default size.

* Click the Close (![Btn-RightPanel-Close.png](Btn-RightPanel-Close.png){width="25" style="inline"}) button to close a panel.

## The Brain Monitor Panel {% #BrainMonitorPanel %}

This panel displays data to show how Amelia processes a conversation. Data is live and updated as a conversation evolves.

*The Brain Monitor Panel*

| Comprehension | Displays which internal system responded during a conversation.<br><br>**SRL**: Displays data from the statistical relational learning system.<br>**ONT**: Displays data for the ontology system.<br>**DNN**: Displays deep neural network system data. |

## The Comprehension Panel {% #ComprehensionPanel %}

This panel displays intents and entities. When triggered, includes data from the Semantic Receptor, Disambiguation, and Sentiment modules.

*The Comprehension Panel*



## The Flow Panel {% #FlowPanel %}

Displays a panel with the current conversation process flow. The Digital Employee Builder (DEB) view is the default. View the BPN process by clicking the BPN button at the top right of the Flow panel.

![B02-00_0002-ChatUI-Flow-Panel.png](B02-00_0002-ChatUI-Flow-Panel.png){width="800"}

## The Journey Analytics Panel {% #JourneyAnalyticsPanel %}

## The Transcripts Panel {% #TranscriptsPanel %}

This panel lets you upload a conversation transcript to replay it. You can download transcripts by clicking the three vertical dots left of the chat input box.

Once uploaded, buttons appear at the top of this panel. Click the Play button to play the transcript. Or click the Upload File button to upload another transcript.

![B02-00_0002-ChatUI-Transcripts-Panel.png](B02-00_0002-ChatUI-Transcripts-Panel.png){width="800"}

*The Transcripts Panel*
